# AIon-Copilot

## 概要

- SlackのようなチャットUIと、GitHub CopilotのようなAIサイドパネルを組み合わせることで、開発者が会話内容をそのままAIに相談できる体験を実現したアプリケーションです。
- SlackやDiscordなどのチャットツールで「この会話の内容がよく分からない」と感じたときに、会話内容をコンテキストとしてAIに質問することができます。
- CursorやGitHub Copilotのように、画面右側にAIチャットパネルが表示されます。
- 任意のメッセージやスレッドをコンテキストとして選択することで、Copy & Pasteや他アプリへの遷移を行うことなく、会話の流れを踏まえた回答をAIが生成します。

## メイン機能

- **AI Copilotチャット**
  - チャット内容やスレッドをコンテキストとしてAIに質問できる
  - 会話の流れを理解した回答を生成

## 工夫点

### マイクロサービスアーキテクチャ

AI推論、チャット処理、APIゲートウェイなどをサービスごとに分離し、拡張性の高い構成にしました。  
サービスごとに独立して開発・スケールできる構成になっています。

### gRPCによる高速な通信

チャット処理とフロントエンド間通信に **Protocol Buffers + gRPC** を採用し、型安全かつ低レイテンシな通信を実現しています。  
Protocol BuffersによってAPIのスキーマを定義することで、フロントエンドとバックエンド間で型を共有しながら安全にデータ通信を行うことができます。

### コンテキストベースAI回答

ユーザーが選択したチャットやスレッドをコンテキストとして整理し、AIへのプロンプトに組み込むことで、単発の質問ではなく**会話の流れを理解した回答**を生成できる仕組みを実装しました。

### Vertex AIを用いたAI推論

Google Cloud **Vertex AI** を利用してLLM推論を行い、スケーラブルなAI処理基盤を構築しています。

### Kubernetesによるスケーラブルなデプロイ

各サービスをDockerコンテナとして構築し、Kubernetes上で管理することで、サービス単位でのスケーリングや運用を可能にしています。

## おまけ（UIのこだわり）

チャット画面はCopilot機能のデモとして実装したものですが、せっかくなのでエンジニアが好きそうな機能をを追加しました。

- ダークテーマ
- キーボードショートカット
- カーソル追従エフェクト

左サイドバー（チャンネルパネル）と右サイドバー（Copilotパネル）は、VSCodeと同様のショートカットで開閉できるようにしたかったのですが、Chrome標準ショートカットと競合するため、`Cmd`ではなく`Ctrl`キーを使用するように変更しています。
